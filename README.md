# UniWorld App

![UniWorld App Banner](https://via.placeholder.com/1200x400/0A2342/FFFFFF?text=UniWorld+App+-+University+Discovery+%26+Application+Assistant)

## A Global University Discovery and Application Assistant

UniWorld App is a comprehensive mobile application designed to streamline the university discovery and application process for students worldwide. Built with a focus on user experience and rich functionality, it helps users explore universities, track scholarships, manage application tasks, and generate personalized email drafts.

## Features

UniWorld App offers a robust set of features across five main tabs:

### 1. Explore Tab
- **Advanced Search & Filter**: Discover universities using keyword search and filters for country, tuition, scholarships, ranking, program, language, intake, type, and acceptance rate.
- **University List**: Browse universities with key information like name, flag, ranking, fee range, and scholarship count.
- **University Detail Screen**: In-depth view of each university, including stats, information sections, scholarships offered, and application deadlines.
- **Save & Compare**: Bookmark universities to "My List" and add them to a side-by-side comparison tool.

### 2. Scholarships Tab
- **Scholarship Listings**: Browse a comprehensive list of scholarships with filters for country, amount range, deadline, and field.
- **Urgency Badges**: Color-coded badges indicate the urgency of application deadlines (red for < 30 days, orange for < 90 days).
- **Scholarship Detail Screen**: View detailed information about each scholarship and bookmark them for later.

### 3. My List & Comparison Tab
- **Saved Universities**: Access a personalized list of saved universities with options to view details, draft emails, or remove from the list.
- **University Comparison**: Compare up to three universities side-by-side, highlighting key differences in fees, ranking, scholarships, and acceptance rates.

### 4. Calendar Tab
- **Deadline Calendar**: Visualize application deadlines for saved universities in a month view, with color-coded urgency.
- **Deadline List**: View a list of upcoming deadlines, with tap-to-view-details functionality.

### 5. Checklist Tab
- **Document Checklist**: Manage application documents with an interactive checklist.
- **Progress Indicator**: Track progress with a clear indicator showing completed vs. pending documents.

### Email Draft Generator
- **Professional Templates**: Generate pre-written, professional email drafts for university inquiries.
- **Customizable Fields**: Easily customize placeholder fields like `[Your Name]`, `[Desired Program]`, and `[Intended Intake Year]`.
- **Quick Actions**: Copy email addresses, copy full email content, or open drafts directly in Gmail.

## Tech Stack

- **Framework**: React Native (Expo)
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS)
- **Database**: Drizzle ORM (for local data persistence, likely SQLite)

## Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mr-10/uniworld-app.git
   cd uniworld-app
   ```
2. **Install dependencies**:
   ```bash
   pnpm install
   # or npm install / yarn install
   ```
3. **Start the Expo development server**:
   ```bash
   pnpm start
   # or npm start / yarn start
   ```
4. **Open on your device/emulator**: Use the Expo Go app on your mobile device or an emulator to scan the QR code.
